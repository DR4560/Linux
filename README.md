## Установка (Linux)
You must have Linux installed [ website]([(https://ubuntu.com/)])

1. Location 

```ls```

2. Change dir

```cd Oxygen```

3. Make dir

```mkdir```

4. Remove dir

```rmdir```

5. Create empty file

```touch```

6. Remove file

```rm```
